Use this application to improve your knowledge in in an Eastern language.

Here you can add decks with cards that symbolise what you want to learn. For example: add a deck with the first character set of the hiragana, katakana or kanji alphabet.

Once you added your decks, it's time to test what you've learned. The test is simple, just select the decks you want to test and inform how long the test lasts, with this done the application will shuffle the pre-registered cards and you will have to guess the correct spelling of the respective symbol.

Hey! You know that you can also register others oriental symbols like korean or chinese alphabets, right?
